## To-Do List con Python y Tkinter

Una aplicación de lista de tareas sencilla y funcional desarrollada en **Python**, utilizando la biblioteca **Tkinter** para la interfaz gráfica y **SQLite** para el almacenamiento de datos.

---

## Características

- Añadir tareas mediante un campo de texto y un botón.
- Marcar tareas como completadas usando `Checkbutton`.
- Eliminar tareas específicas con un botón de borrado.
- Persistencia automática de datos usando **SQLite** (las tareas se guardan y se cargan al iniciar la aplicación).

---

## Tecnologías Utilizadas

- Python 3
- Tkinter – para la interfaz gráfica
- SQLite – para almacenamiento local de tareas
